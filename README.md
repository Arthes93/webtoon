# 프로젝트 소개
Spring Boot 2.1 기반의 토이 프로젝트입니다. 많은 웹툰 플랫폼들이 있는데 이런 플랫폼들을 한눈에 볼 수 있게끔 만들 계획입니다.
(작성중)

# 기능
- 요일별 웹툰 목록 출력.
> 1. 네이버
> 1. 다음
- 로그를 활용한 전체 플랫폼범위의 인기차트순위 출력.
- 웹툰 목록 사용자화 가능
- 추가기능 구상 중..

# 프로젝트 소개
다양한 플랫폼에서 많은 웹툰을 찾아보는 사람들을 위해 만들게 되었습니다. 이 프로젝트는 다양한 웹툰 플랫폼들의 웹툰 목록들을 한데 모아 볼 수 있는 서비스이며 부가적인 기능들을 추가적으로 구상 중입니다.

# 프로젝트 스펙
- Spring Boot 2.1
- Spring Data Jpa

# 기능
- 요일별 웹툰 목록 출력.(네이버, 다음 등)
- 소셜 로그인
- 유저별 웹툰 목록 커스터마이징
- 구상중..
